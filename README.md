# Mi Portal de Empleo

Plataforma minimalista para conectar candidatos y ofertas en Chile. Diseñada para postular sin registros y con un flujo directo.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Estructura

- `src/pages/` páginas principales
- `src/components/` componentes reutilizables
- `src/lib/` datos y utilidades
- `src/styles/` estilos globales
